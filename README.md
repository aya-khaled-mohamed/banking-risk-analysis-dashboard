# 📊 Banking Risk Analysis Dashboard

An interactive Power BI dashboard designed for comprehensive banking risk assessment, loan performance tracking, customer segmentation, and financial insights. This project helps financial institutions leverage data-driven strategies to mitigate risks, optimize loan portfolios, and enhance customer retention.

---

## 📌 Project Overview
In the banking sector, managing risk and understanding loan performance is critical for financial stability. This repository contains a complete Power BI solution that transforms raw financial data into actionable business intelligence insights. 

The dashboard provides a deep dive into key metrics such as non-performing loans (NPLs), credit risk categories, default rates, and customer demographic segmentation.

---

## 📂 Repository Structure
As shown in the project repository structure (refer to `image_66149d.jpg`), the files are organized as follows:

*   **`Data/`**: Contains the datasets used for the analysis.
*   **`Icons/`**: Design assets and UI icons used to build a polished, professional interface.
*   **`Wallpaper/`**: Custom backgrounds and canvas layouts designed for the dashboard pages.
*   **`Banking.pbix`**: The main Power BI desktop file containing data models, DAX measures, and visualizations.
*   **`Banking.pdf`**: A static, high-quality export of the final dashboard for quick review and presentations.

---

## 🚀 Key Features & Insights
*   **Risk Assessment Matrix**: Visualizes risk distribution across different loan types and customer brackets to identify high-risk areas.
*   **Loan Performance Tracking**: Comprehensive KPIs evaluating total active loans, default rates, recovery rates, and overdue payments.
*   **Customer Segmentation**: Dynamic slicing of banking clients based on credit scores, income brackets, age groups, and geographical distribution.
*   **Financial Insights**: Trend analysis over time to spot seasonal patterns in loan applications and repayment behaviors.

---

## 🛠️ Tools & Technologies Used
*   **Power BI Desktop**: For ETL (Data Extraction, Transformation, and Loading), Data Modeling, and interactive Data Visualization.
*   **Power Query**: Used for robust data cleaning, handling missing values, changing data types, and merging tables.
*   **DAX (Data Analysis Expressions)**: Leveraged to build complex dynamic calculations, time-intelligence functions, and custom business KPIs.
*   **UI/UX Design Tools**: For creating clean, thematic wallpapers and custom asset layouts to maximize dashboard readability.
